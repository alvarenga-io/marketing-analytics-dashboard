## Dicionário de Dados

A base de dados consiste em uma tabela única contendo o histórico de campanhas de marketing. Abaixo, a descrição de cada coluna:

| Coluna | Tipo de Dado | Descrição |
| :--- | :--- | :--- |
| **ID** | Int  | Número de registro individual do cliente | 
| **Ano Nascimento** | Int  | Ano do nascimento do cliente | 
| **Escolaridade** | String | Grau de escolaridade |
| **Estado Civil** | String | Estado civil | 
| **Salario Anual** | Int | Soma dos salários recebidos em um ano |
| **Filhos em Casa** | int | Número de crianças pequenas em casa |
| **Adolescentes em Casa** | Int | Número de adolescentes em casa |
| **Data Cadastro** | Date | Data do regristro do cliente|
